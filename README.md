# Darden Recruiting OS

React/Vite app with Events, Networking CRM, Applications, and Tasks.

Seeded events: 29

## Run
```bash
npm install
npm run dev
```

## Included event fields
- date, start/end time, timezone
- registration/portal link
- Zoom/Teams/Meet link
- passcode
- source email URL
- status and notes

BCG portal event is included as manual-entry placeholder.
